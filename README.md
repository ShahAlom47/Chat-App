# Next.js Chat Application

## Overview
A real-time chat application built with Next.js, featuring modern web technologies and responsive design.

## Features
- Real-time messaging
- User authentication
- Responsive UI
- Modern tech stack

## Installation
```bash
npm install
npm run dev
```

## Getting Started
1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Run the development server at `http://localhost:3000`

## Tech Stack
- Next.js
- React
- Node.js
- WebSocket/Socket.io (for real-time features)

## Project Structure
```
├── pages/
├── components/
├── public/
└── styles/
```
